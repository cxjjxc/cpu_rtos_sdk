## 概述

FreeRTOS是一个迷你的实时操作系统内核。作为一个轻量级的操作系统，功能包括：任务管理、时间管理、信号量、消息队列、内存管理、记录功能、软件定时器、协程等，可基本满足较小系统的需要。

## 功能和特点

用户可配置内核功能

多平台的支持

提供一个高层次的信任代码的完整性

目标代码小，简单易用

遵循MISRA-C标准的编程规范

强大的执行跟踪功能

堆栈溢出检测

没有限制的任务数量

没有限制的任务优先级

多个任务可以分配相同的优先权

队列，二进制信号量，计数信号灯和递归通信和同步的任务

优先级继承

免费开源的源代码
